# Heart Disease Prediction Project

## Overview
This project predicts the presence of heart disease based on medical attributes such as age, sex, chest pain type, blood pressure, cholesterol levels, and other health indicators using Machine Learning classification models.

## Features
- Predict heart disease (Yes/No)
- Data preprocessing and cleaning
- Model training and evaluation
- Beginner-friendly ML classification project

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Pickle
